Contrast Demo

Download and Install CMake

You can get it from here: 
   http://www.cmake.org/cmake/resources/software.html
Next, open a command prompt, go to the source folder for this demo, and perform these actions: 
   > mkdir build
   > cd build
   > cmake .. -G "Visual Studio 8 2005"  
   > Contrast.sln
   > Right-click "Contrast" in Solution Explorer and make it your StartUp Project.
   > Press F5
